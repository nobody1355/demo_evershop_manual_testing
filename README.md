## Documentation

### Overview

This project tests the functional accuracy and API consistency of key features of an e-commerce website. Testing covered search behaviour, product details, variations, add-to-cart, cart viewing, checkout workflows.

### Work Done

* Created requirement analysis questions for the client.

* Designed a structured test suite, including 24 detailed test cases for search and product variation flows.

* Executed UI functional tests and logged defects.

* Performed API testing for product search, product details, cart creation, and checkout.

* Reported API-UI mismatches in cart responses, and variation mapping.

* Created supporting documentation: test suite, defect log, comparative analysis, analysis feedback, and sign-off report.
